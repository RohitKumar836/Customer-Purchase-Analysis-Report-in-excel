**Customer Purchase Analysis Dashboard


**📌 Project Overview****

This project focuses on analyzing customer purchase behavior using Excel and building an interactive dashboard to derive meaningful business insights.

**🎯 Objectives**
Analyze customer demographics and purchasing trends
Identify top-performing categories and products
Understand seasonal and behavioral patterns
Build an interactive dashboard for decision-making
**📂 Dataset Features**
Customer demographics (Age, Gender)
Product details (Category, Size, Color)
Sales data (Purchase Amount, Season)
Customer behavior (Frequency, Subscription)
Transaction details (Payment Method, Shipping Type)
**📊 Key Insights**
Total Customers: 3,900
Total Revenue: $233,081
Average Rating: 3.8
Clothing is the top revenue-generating category ($104,264)
Top products: Blouse, Shirt, Dress, Pants
Male customers dominate purchases
Age group 50+ contributes the highest revenue ($88,480)
Fall season shows peak sales ($60,018)
Medium size and Green color are most preferred
Credit Card is the most used payment method
**📈 Dashboard Features**
KPI Cards (Revenue, Customers, Rating)
Category & Product Analysis
Customer Demographics Analysis
Seasonal Trends
Interactive Slicers for filtering
Tools Used
Microsoft Excel
Pivot Tables
Data Visualization
Dashboard Design
